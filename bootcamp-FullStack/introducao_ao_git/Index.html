Introduçao ao Git/GitHub

Comandos no Bash

  • Ls - Lista de todas as pastas e arquivos
  • Cd - Entra na pasta
  • Cd .. - Volta um nível na pasta
  • mkdir - Cria pasta
  • echo hello > hello.txt - Cria um arquivo txt com a palavra "Hello"
  • rm - deleta os arquivos
  • rmdir - deleta pastas

Iniciando Git

  • git init
  • git config --global user.email “jean.lcn12@gmail.com”
  • git config --global user.name “JeanLCN”
  • git add *
  • git commit -m “commit inicial(msg commit)”
  • git add - adicionar arquivos que sofreram modificações
  • git remote add origin “https://github.com/JeanLCNs/livro-receitas.git” (link do repositório) - adicionar repositório remoto
